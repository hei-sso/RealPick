# Real Pick

## 📘 개요
Do Or Pay는 사용자가 습관 목표를 설정하고 포인트를 걸어 인증하는 모바일 애플리케이션입니다.\
목표 성공 시 포인트를 반환하고, 실패 시 포인트가 차감되며, 그룹 챌린지에서는 실패자의 포인트가 성공자에게 분배됩니다.

## 🧩 주요 기능
- 광고성 키워드 탐지 및 반복 문구 패턴 분석을 통해 0~100점 사이의 신뢰도 점수 산출
- 리뷰를 분석하여 긍정/부정 핵심 요약 및 자주 언급되는 장단점 추출
- 광고성 리뷰를 제외한 실제 사용자의 리뷰만을 기반으로 제품 평점 재산출
- 별도의 사이트 이동 없이 쇼핑몰 제품 페이지 내에서 분석 카드 형태로 결과 표시

## 📦 기술 스택
- **Frontend**: React, TypeScript, Chrome Extension (Manifest V3)
- **Backend/Server**: Node.js (Express), Railway
- **AI/ML**: OpenAI API
- **DB**: MySQL
